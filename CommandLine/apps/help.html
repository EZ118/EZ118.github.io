有关某个命令的详细信息，请键入 HELP 命令名
CD             显示当前目录的名称或将其更改。
CLS            清除屏幕。
COLOR          设置默认控制台前景和背景颜色。
DATE           显示或设置日期。
DIR            显示一个目录中的文件和子目录。
               获取文章列表。
ECHO           显示消息，或将命令回显打开或关闭。
EXIT           退出 博客系统。
FIND           在一个或多个文件中搜索一个文本字符串。
HELP           提供 博客系统 命令的帮助信息。
PAUSE          暂停批处理文件的处理并显示消息。
PRINT          打印一个文本文件。
SET            显示、设置或删除 博客系统 配置变量。
SHUTDOWN       允许通过本地或远程方式正确关闭计算机。
TIME           显示或设置系统时间。
TITLE          设置 博客系统 会话的窗口标题。
TYPE           显示文本文件的内容。
VER            显示 博客系统 的版本。

有关工具的详细信息，请参阅联机帮助中的命令行参考。
